  • In my masters' course I took courses in Machine Learning, Artificial Intelligence,Cloud Computing and Distributed Computing.

  • Wanting to delve a bit deeper beyond the course material and text book knowledge, I started doing The Machine Learning Course by Andrew Ng on Coursera.
  • The first opportunity to try something hands-on came with the Intuit Women’s Code War conducted by Hackerrank where a friend of mine and I started learning Python on the go,(deployed APIs in Flask and built the UI in React) and created an intelligent tax management platform using Machine Learning Concepts. The idea was shortlisted.
  • Having seen my interest in this field, I was given the next assignment from work which is to "Predict who is next likely to login to our system?".I created a preliminary prototype by analysing the data we have in our system using time series analysis methodologies and the refinements of the model is still in progress.
